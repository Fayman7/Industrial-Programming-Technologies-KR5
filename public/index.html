API Дневник настроения

Возможности:

GET - Получить все записи
/api/moods

Query параметры: ?date=2024-01-15, ?mood=happy, ?tag=учеба

GET - Получить запись по ID
/api/moods/1
POST - Создать новую запись
/api/moods

Формат запроса(строка): {"date": "2024-01-18", "mood": "happy", "description": "...", "tags": ["тег1", "тег2"]}

PUT - Обновить запись
/api/moods/1
DELETE - Удалить запись
/api/moods/1
GET - Статистика
/api/moods/stats/overview
